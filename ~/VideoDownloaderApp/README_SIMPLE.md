## 🚀 **Kaise Use Karein:**

### **Step 1: Android Studio Install Karein**
1. Android Studio download karein: https://developer.android.com/studio
2. Install karein aur open karein

### **Step 2: Project Open Karein**
1. Android Studio mein "Open an existing project" click karein
2. `~/VideoDownloaderApp` folder select karein
3. Wait karein Gradle sync ke liye

### **Step 3: Build Karein**
1. **Build** → **Build Bundle(s) / APK(s)** → **Build APK(s)**
2. APK generate hoga `app/build/outputs/apk/debug/` mein

### **Step 4: Install Karein**
1. APK file phone mein copy karein
2. Install karein (Unknown Sources enable karein)
3. App open karein aur video download karein!
